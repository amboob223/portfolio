What is the Wocktan?

"I have developed a professional language learning web application called Wocktan, derived from the Wolof word 'waxtan,' which means conversation. Utilizing React.js, this application features a user-friendly form that facilitates API requests to a secure backend server. The server efficiently stores the collected data in a spreadsheet, ensuring effective organization. The main objective of Wocktan is to bridge the gap between native language speakers and individuals who are eager to learn indigenous African languages.

Furthermore, Wocktan leverages the power of React components, including a custom-designed scroller component that provides a seamless language display. To enhance usability, I have implemented a dropdown menu using React, allowing users to navigate through the site effortlessly. Additionally, I have seamlessly integrated the web application with the metaverse through my Digital Wocktan School on the Decentraland platform. You can access the platform at the specific address -60, -18.

By combining cutting-edge technology, intuitive design, and integration with the metaverse, Wocktan offers a professional and immersive experience for users passionate about learning indigenous African languages."
